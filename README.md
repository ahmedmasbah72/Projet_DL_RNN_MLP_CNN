# Projet_DL_RNN_MLP_CNN
Deep Learning Training Reference Guide is an end-of-year engineering project that implements a complete pipeline for training and comparing several neural network architectures (MLP, CNN, RNN, LSTM, GRU) on heterogeneous data: tabular (Breast Cancer Wisconsin), images (MNIST), and specialized text corpora. The project is built in PyTorch and focuses on data preparation, model design, hyperparameter tuning, and rigorous evaluation (accuracy, F1‑score, confusion matrices, perplexity), with detailed documentation of experiments and training strategies (GPU usage, gradient clipping, teacher forcing, regularization techniques). It serves as a practical reference for students and practitioners who want to understand how to choose and train deep learning architectures according to the nature of the data and the task.
